# AWESOMECRAFT Store Page Draft

## Short Description

AWESOMECRAFT is a playful block-building survival sandbox where you create worlds, craft gear, explore portals, and battle bosses from classic or first-person view.

Awesome Development access price: 7 ADC.

## About This Game

Build your own world, survive the night, jump between dimensions, and face big boss fights in AWESOMECRAFT. Start a new survival or creative world, customize your player, gather blocks, craft with a 3x3 menu, and discover computers with the BuildTube app.

Explore the Overworld, enter the Nether, travel to the End, and challenge the Ender Dragon. You can switch between Classic view and First Person mode any time, with turnable controls for keyboard and touch.

## Features

- New World and Load World menus
- Creative and Survival modes
- Classic view and turnable First Person mode
- Inventory with 3x3 crafting
- Day and night cycle with monsters
- Nether and End dimensions
- Giga Slime and Ender Dragon boss fights
- Player creator
- Computer blocks with BuildTube
- Local realm-style multiplayer sync
- Home-screen install support for the web build
